# History

## Purpose
Keep the origin, evolution, milestones, and legacy of the project in one place.

## Files
- `ORIGIN.md` — earliest source of the project.
- `EVOLUTION.md` — progression from early pieces to canon.
- `LEGACY.md` — long-term decisions and lessons.
- `MILESTONES.md` — major turning points.

## Rule
History records the path taken by the project; it does not replace canon.
