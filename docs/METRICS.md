# Metrics

## Purpose
Measure how the system is performing so SinergYa can improve creative quality, production efficiency, and canon stability over time.

## Metrics
- Time from idea to brief
- Time from brief to prompt
- Time from prompt to validated piece
- Number of revisions per piece
- Validation pass rate
- Canon break rate
- Merchandising suitability rate
- 3D suitability rate
- Parody recognizability rate
- Percentage of pieces archived as reference

## Use
Metrics are for learning, not bureaucracy.
They help the team identify friction, strengths, and recurring failure modes.

## Suggested reporting cadence
- Per piece
- Per collection
- Per month
- Per quarter
