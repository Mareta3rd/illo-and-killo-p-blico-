# Roadmap

## Estado del proyecto
Illo & Killo se desarrolla como base de contenido, canon y laboratorio de experimentación para SinergYa.

## Fases
### Fase 1 · Canon
- Illo y Killo cerrados
- Biblia y validación

### Fase 2 · Biblioteca
- Objetos oficiales
- Fauna oficial
- Patrimonio oficial

### Fase 3 · Producción
- Gags canonizados
- Parodias culturales
- Merchandising
- 3D

### Fase 4 · Automatización
- Prompt compilado
- Validación asistida
- Flujo de generación

### Fase 5 · Expansión
- Nuevas colecciones
- Nuevos formatos
- Nuevos universos basados en el mismo sistema

## Prioridad actual
Cerrar el canon, estabilizar el sistema y comenzar la experimentación controlada con contenido de Illo & Killo.
