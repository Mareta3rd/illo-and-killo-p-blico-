# Principles

## Purpose
These principles guide all creative and operational decisions in SinergYa.

## Principles
- Simplicity beats excess detail.
- Gag beats decoration.
- Recognition beats forced originality.
- Consistency beats isolated beauty.
- Collection beats single-piece ego.
- Experimentation is allowed inside the lab, not inside the canon.
- The study’s memory must preserve decisions, not only results.
- Illo & Killo is the first content base for structured experimentation.

## Use
Whenever there is doubt, consult these principles before adding new rules.
