# Schema

## Objetivo
Convertir el conocimiento de Illo & Killo en datos estructurados para facilitar automatización, validación y generación asistida.

## Entidades principales
- character
- gag
- object
- fauna
- heritage
- palette
- workflow
- prompt
- decision

## Propósito
Este esquema permite que el sistema consulte reglas y contenidos sin depender sólo de texto libre.

## Uso previsto
- validación asistida
- compilación de prompts
- trazabilidad de decisiones
- exportación a otros formatos
- experimentación controlada
