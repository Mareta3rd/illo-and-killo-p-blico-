# Paleta oficial

## Principio
Los colores no describen la realidad: describen el universo Illo & Killo.

## Base
- Blanco limpio para Illo.
- Rojo coral o magenta contenido para Killo.
- Negro dominante para contorno y contraste.
- Verde para el pañuelo de Illo.
- Amarillo ácido para la mecha.
- Turquesa, cian o lima como acento según el gag.

## Reglas
- No usar degradados salvo necesidad excepcional.
- No usar paletas naturales si rompen el canon.
- Cada pieza debe parecer impresa o serigrafiada, no fotográfica.
- El color de apoyo nunca debe competir con el gag.
