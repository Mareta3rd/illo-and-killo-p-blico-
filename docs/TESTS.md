# Tests

## Purpose
Evaluate whether a proposed piece, prompt, or workflow respects the Illo & Killo system.

## Test areas
- Canon of characters
- Humor quality
- Cultural recognition
- Composition clarity
- Color consistency
- Merchandising viability
- 3D viability
- Workflow fit
- Prompt completeness
- Data consistency

## Output
Each test returns:
- PASS
- PASS WITH NOTES
- FAIL
- FAIL WITH REASONS
