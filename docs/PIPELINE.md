# Pipeline de producción

## Flujo general
1. Proponer una idea.
2. Clasificarla: gag original, parodia cultural, homenaje o variación de biblioteca.
3. Seleccionar canon de personajes.
4. Seleccionar escenario.
5. Seleccionar objetos oficiales.
6. Seleccionar fauna o patrimonio si procede.
7. Redactar el prompt específico.
8. Generar la pieza.
9. Validar contra el checklist.
10. Canonizar, corregir o descartar.
11. Exportar a destino: web, impresión, merchandising o 3D.

## Regla
Ninguna pieza pasa a producción sin validar canon, humor y composición.
