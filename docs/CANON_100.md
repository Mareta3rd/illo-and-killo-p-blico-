# CANON 100

## Objetivo
Este documento define la identidad inmutable de Illo & Killo.

## Principios
- El canon prevalece sobre cualquier ilustración individual.
- Ninguna IA ni ilustrador reinterpretará libremente a los personajes.
- Toda nueva pieza deberá ser consistente con el canon.

## Illo
- Optimista hasta el absurdo.
- Criatura original, no un caballo.
- Silueta reconocible.
- Hocico integrado.
- Mechón amarillo en forma de llama.
- Pañuelo verde.
- Cola corta terminada en una pequeña llama.

## Killo
- Prudente y reactivo.
- Criatura original, no un toro.
- Cuerpo compacto.
- Clavel en la oreja.
- Lunares negros.

## Validación
Cada nueva ilustración deberá superar:
1. Test de silueta.
2. Test de expresión.
3. Test del gag.
4. Test de segunda lectura.
5. Test de consistencia.
