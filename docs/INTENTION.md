# Intention

## Core principle
Every element included in a piece must have a clear intention.

If its purpose cannot be explained, it should normally be removed.

## Valid intentions
- Main gag
- Secondary reading
- Cultural reference
- Character identity
- Composition
- Narrative
- Production
- Collection continuity
- Merchandising

## Character invariants
### Level A – Invariants
These should not change except by explicit documented decision.

Examples:
- Killo's clavel in the ear.
- Canonical silhouettes.
- Core personality dynamics.

### Level B – Adaptable
Clothing, profession, historical period, props.

### Level C – Variable
Background, scene details, secondary characters, typography, atmosphere.

## Asset reuse
Recurring assets (e.g. mosquito, shark, ham, espeto) should only be reused when there is explicit narrative or comedic intention. Repetition alone is not a reason.

## Cultural transduction
Parodies should translate the essence of the source into the Andalusian and Illo & Killo universe, not merely replace backgrounds.
