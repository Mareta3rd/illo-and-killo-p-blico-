# State

## Purpose
Track the lifecycle of each asset, decision, prompt, or experiment in the SinergYa system.

## States
- Idea
- Lab
- Prototype
- Validation
- Canon
- Production
- Archived

## Usage
- Idea: a concept not yet routed.
- Lab: an experimental piece that may fail safely.
- Prototype: a first usable version.
- Validation: under tests and review.
- Canon: approved and stable.
- Production: ready for output and commercialization.
- Archived: kept for reference but not active.

## Rule
Anything affecting canon must be clearly marked with its current state.
