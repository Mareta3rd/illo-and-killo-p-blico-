# Workflow · Character

## Objetivo
Cerrar un personaje canónico sin reinterpretaciones libres.

## Pasos
1. Partir del canon visual maestro.
2. Describir proporciones en términos medibles.
3. Fijar rasgos invariantes.
4. Fijar rasgos secundarios.
5. Definir límites.
6. Validar silueta.
7. Validar expresiones.
8. Validar poses.
9. Canonizar la versión final.
