# Illo & Killo

Repositorio de automatización del flujo creativo para el universo Illo & Killo.

## Propósito
Este repositorio actúa como base de trabajo para:
- Canon visual
- Biblia narrativa
- Model sheets
- Gags canónicos
- Objetos, fauna y patrimonio simplificados
- Preparación de assets para 3D y merchandising
- Automatización del proceso creativo

## Principios de trabajo
- Illo y Killo son los personajes oficiales y canónicos.
- Ninguna generación debe reinterpretarlos libremente.
- Toda nueva pieza debe respetar el canon aprobado.
- La referencia maestra es la ilustración canónica consolidada.
- El criterio principal es la consistencia entre piezas.
