# Validate piece

## Objective
Evaluate a generated piece against the canon and the production rules before accepting it.

## Inputs
- character
- gag
- composition
- palette
- background
- objects
- fauna
- heritage
- support
- destination

## Checks
1. Character canon.
2. Silhouette recognition.
3. Gag clarity.
4. Second reading.
5. Cultural recognition if applicable.
6. Production viability.
7. Merchandising viability.
8. 3D viability if applicable.
9. No unnecessary clutter.
10. No canon break.

## Output
- Pass
- Pass with notes
- Fail
- Fail with reasons
