# Generate brief

## Objective
Convert an idea into a complete creative brief before prompt compilation.

## Inputs
- idea
- route
- target
- cultural_reference
- support
- destination
- tone
- constraints

## Output sections
- title
- summary
- intent
- must-keep elements
- optional elements
- risks
- required assets
- production notes
- validation focus

## Rule
The brief must be based on approved canon, workflows and data. It must not invent new canon.
