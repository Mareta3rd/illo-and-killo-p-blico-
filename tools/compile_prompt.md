# Compile prompt

## Objective
Assemble a production prompt from structured Illo & Killo data.

## Inputs
- character
- gag
- objects
- fauna
- heritage
- palette
- destination
- support

## Steps
1. Load the selected character canon.
2. Load the gag or creative intent.
3. Load the required objects.
4. Load fauna or heritage only if needed.
5. Apply palette and composition rules.
6. Apply the destination constraints.
7. Build a prompt that respects the canon and the workflow.
8. Return the final prompt text.

## Rule
The compiler must not invent canon; it must only assemble and adapt approved knowledge.
