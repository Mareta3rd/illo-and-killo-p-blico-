# Prompt de biblioteca de objetos

Selecciona objetos oficiales del universo Illo & Killo para una nueva pieza. No inventes versiones alternativas si el objeto ya existe en la biblioteca.

## Reglas
- Reutilizar objetos canónicos.
- Simplificar al lenguaje visual del universo.
- Mantener reconocimiento inmediato.
- No recargar la escena con objetos innecesarios.
- Si un objeto no aporta al gag, se elimina.
