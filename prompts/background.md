# Prompt de fondo

Genera un fondo secundario para una pieza de Illo & Killo. El fondo debe reforzar el gag sin competir con los personajes.

## Reglas
- Usar pocos elementos.
- Preferir siluetas simples.
- Si aparece patrimonio, usar su versión canónica simplificada.
- Si aparece fauna, usarla sólo como apoyo narrativo.
- Evitar saturación visual.
- El fondo debe respirar y dejar protagonismo a Illo y Killo.
