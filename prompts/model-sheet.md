# Prompt de model sheet

Genera una hoja de modelo canónica de Illo o Killo. No reinterpretar. No estilizar libremente. Reconstruir exactamente al personaje oficial.

## Reglas
- Usar la referencia canónica como verdad absoluta.
- Mantener proporciones, silueta, hocico, patas, manos, orejas, cola y pañuelo según el canon.
- Mostrar vistas útiles para producción: frontal, perfil, tres cuartos y espalda.
- Las expresiones deben ser mínimas y consistentes.
- El objetivo no es embellecer, sino documentar.
